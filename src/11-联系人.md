## 升级路径

```
一线运维 → 运维主管 → 项目经理 → 客户负责人
   │           │           │           │
 15分钟     30分钟      1小时      2小时
```

**升级原则**：
- P0故障：立即升级至项目经理
- P1故障：30分钟未解决升级至运维主管
- P2故障：2小时未解决升级至运维主管
